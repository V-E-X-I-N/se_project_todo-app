# Simple Todo App

A clean, Todo application built with vanilla JavaScript.
Users can add tasks, mark them as complete, view due dates, and remove tasks from the list.
Project focuses on clear separation of data, DOM rendering, and user interaction.

## Functionality

- Add new todo items using a form.
- Assign and display a due date for each task.
- Mark tasks as completed using a checkbox.
- Remove tasks from the list.
- Form validation with real-time feedback.
- Submit button enables/disables based on input validity.
- Initial todos are rendered on page load.

## Technology

- **HTML** - semantic structure and templates
- **CSS** - layout and visual styling.
- **JavaScript**
  - Classes for component-based structure.
  - Modules (import / export) for separation JS files.
  - DOM manipulation using querySelector and templates.
  - Event handling for user interactions.
- **UUID** - unique identifiers for each todo item.
- **Git & GitHub** - version control and project hosting.
